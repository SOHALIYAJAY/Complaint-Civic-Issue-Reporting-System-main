🏙️ Complaint Civic Issue Reporting System

A full-stack web application that enables citizens to report civic issues and allows authorities to manage and resolve them efficiently.

📌 Overview

The Complaint Civic Issue Reporting System provides a centralized platform where users can report issues like potholes, garbage, water leakage, etc., and track their resolution status in real-time.


👥 Users

Citizens → Raise & track complaints

Officers → Manage and update complaints

Admins → Monitor system & manage departments
